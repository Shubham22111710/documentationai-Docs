<Update label="2024-11-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added production scheduling dashboard with Gantt chart views for TMT bars and steel tube manufacturers.
- Introduced pharma stockist module for batch tracking and expiry date alerts.
- Enhanced metal fabrication tools with real-time material yield calculations.

## Improvements

- Optimized inventory queries for faster performance during peak production hours.
- Improved mobile responsiveness for safety shoes and footwear production logs.

## Bug Fixes

- Fixed sync issues between finance and production modules.
- Resolved export errors for channel and section manufacturing reports.

</Update>

<Update label="2024-10-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched fasteners and forging unit integration with automated quality control checklists.
- Added strips and coil processing workflow with multi-stage approval routing.
- Integrated real-time webhook support for external inventory systems.

## Breaking Changes

## Bug Fixes

- Fixed login timeouts during high-volume pharma distributor sessions.

</Update>

## Migration Guide

Use these steps to update to the latest version. Backup your data first.

<Steps>
  <Step title="Backup Data" icon="database">
    Export all modules via the dashboard.

````bash
# From your Knowiterp installation directory
knowiterp backup --all-modules
````

  </Step>
  <Step title="Download Update" icon="download">
    Get the latest release from your account portal.
  </Step>
  <Step title="Apply Update" icon="upload">
    Run the updater with your license key.

````bash
knowiterp update --version 2.1.0 --license YOUR_LICENSE_KEY
````

  </Step>
  <Step title="Verify" icon="check-circle">
    Test key workflows like production planning.
  </Step>
</Steps>
